## ✅ Contenido

### 📝 Preguntas de Selección Múltiple

| Pregunta | Tema | Respuesta |
|----------|------|-----------|
| 1 | ¿Qué es Git? | B - Sistema de control de versiones |
| 2 | Función del repositorio | B - Registrar y organizar archivos |
| 3 | Comando `git add` | B - Agregar archivos al área de preparación |
| 4 | Comando `git commit` | A - Guardar registro de cambios |

### 💡 Conceptos Clave

- **Repositorio** → Espacio donde se guardan los archivos y cambios de un proyecto
- **Commit** → Registro de cambios realizado en un momento específico
- **Push** → Enviar cambios desde la máquina local hacia GitHub

### 🤔 Reflexión

> *"Un proyecto con trazabilidad permite saber qué se hizo, cuándo se hizo y cómo evolucionó el trabajo técnico."*
